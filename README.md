Orientações para executar o projeto:

- Após clonar o repositório deve ser executado o comando yarn install ou npm install no terminal ou powerShell abertos dentro do diretório do projeto para instalar todas as dependências do projeto.

-Quando a instalação de dependências terminar, ainda no terminal ou no powerShell deve ser executado o comando yarn start ou npm start para executar o projeto.

-Agora com todas as dependências instaladas e com o projeto rodando para visualizar a aplicação basta acessar a url localhost:3000/ no navegador de sua preferência.
